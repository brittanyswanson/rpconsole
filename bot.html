Commands

Commands are always prefaced with an exclamation mark (!).

command purpose example
species Shows the current available species !species
char_list <species> Shows the active characters of a species !char_list vampire
chars <player name> Shows the active characters for a player !chars "mystery player"
app <character> Shows the application url and player name for a character !app "marty mcfly"
